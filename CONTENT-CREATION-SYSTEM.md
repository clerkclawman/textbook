## How to Use This System

### For Subagents Creating New Content

1. **Identify Target Level** → Read corresponding SPECS-*.md file
2. **Review Grammar Requirements** → Check "Grammar Specifications" section
3. **Check Vocabulary Range** → Use word count and characteristics table
4. **Template Your Content** → Follow universal template format

### Example Workflow

**Task:** Create a new Eiken 3 story

1. Read `SPECS-EIKEN3.md`
2. Identify required grammar: Present perfect, superlatives, conditionals
3. Check vocabulary: 1,500-2,500 words, A2 level
4. Follow template pattern: Emoji + English + Japanese
5. Verify with checklist at end of spec file
