## Test Design Notes

**Answer Quality Assurance:**
- Every "wrong" answer is unambiguously incorrect—either grammatically impossible or semantically nonsensical

- Distractors are designed to test common mistakes at each level

**Level Progression:**
- Section A (1-5): Tests foundational concepts (demonstratives, basic present tense, prepositions)
- Section B (6-10): Tests present continuous, pronouns, and imperatives
- Section C (11-15): Tests past tense, modals, and comparatives
- Section D (16-20): Tests present perfect, passive voice, tag questions, and reading comprehension
